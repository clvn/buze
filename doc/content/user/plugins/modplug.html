Modplug tracker

The Modplug plugin implements the OpenMPT core for playback of .MOD, .S3M, .IT modules. The exposed effect set is the internal OpenMPT effects.

The Modplug tracker is used as the default plugin when importing old-school modules from the "File | Open" menu.

Global parameters

Parameter name Type Description
none

Track parameters

Parameter name Type Description
Note note Note
Wave byte Wavetable index
Volume command byte Volume column command, see table below.
