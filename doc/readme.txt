This directory contains documentation in X(HT)ML and scripts + templates for generating a website and a compiled HTML help file.

Start make_web.wsf to generate themed documentation for the web site.

Start make_chm.wsf to generate themed documentation for HTML Help Workshop.

Use HTML Help Workshop to generate a CHM.
